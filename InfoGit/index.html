Información sobre Git

¿Qué es Git?

Git es un sistema de control de versiones distribuido para rastrear cambios en archivos y coordinar trabajo en proyectos.

Página oficial de Git

¿Qué es GitHub?

GitHub es una plataforma de alojamiento de código que utiliza Git para el control de versiones, facilitando la colaboración entre desarrolladores.
