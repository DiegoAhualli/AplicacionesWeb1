  • Info comandos Git
  • git init
    • Inicializa un nuevo repositorio Git.
  • git clone [url]
    • Clona un repositorio existente desde una URL.
  • git add [file]
    • Agrega archivos al área de preparación.
  • git commit -m "mensaje"
    • Guarda los cambios en el historial de commits.
  • git push
    • Envía los commits al repositorio remoto.
  • git pull
    • Descarga y fusiona los cambios del repositorio remoto.

Ejemplo de clonar un repositorio en nuestro escritorio con una consola

            
            cd Desktop
            git clone https://github.com/DiegoAhualli/AplicacionesWeb1.git
            cd AplicacionesWeb1
